Data Visualization Workshops

Materials from our popular Data Visualization Workshop Series

Getting Started with Data Visualization I

In this introductory workshop to data visualization, participants will gain an understanding of the process for making good visualizations and practice with two tools, Tableau and Raw.

Slides: PPT | Instructions: PDF / DOC

Getting Started with Data Visualization II

In this workshop, participants will have the opportunity to strengthen your understanding of the process for making good visualizations, review core design concepts, learn about various chart types and their best case uses. Participants will also practice doing a visualization ìmake over.î

Slides: PPT | Instructions | Sample Data

Tableau I

Tableau is a free data visualization software. In this introductory workshop to Tableau, participants will learn how to upload a dataset to Tableau, learn about the visualization types available in Tableau, create multiple visualizations and a dashboard including a bar chart, map, line chart, and tree map. You will also learn how to publish a visualization.

Slides: PDF/ PPT | Instructions | Sample Data
Videos:
Renaming Data Columns in Tableau Activity 1: Stacked Bar Chart Activity 2: Map Activity 3: Line Chart Activity 4: Tree Map Activity 5: Dashboard

Tableau II

In this more advanced Tableau workshop, participants will learn more advanced visualization techniques such as filtering, creating and using sets, using multiple measures, selected highlighting, quick table calculatioons, dual axis charts, combination charts, and story points.

Slides: PPT | Instructions Sample Data
Videos:
Activity 1: Easy Bar Chart Activity 2: Map Activity 3: Calculated Field Activity 4: Data Source Filter Activity 5: Combination Chart

Data Cleaning with Open Refine

Open Refine is a free, open source tool that makes it easy to clean and transform large data sets. This workshop will introduce you to how to clean your data to remove problematic characters, whitespaces, symbols, errors, missing elements and inconsistencies.

Slides: PPT / PDF | Instructions

Infographics

In this workshop, participants will explore the difference between infographics and data visualization, learn best practices for infographic design, and practice building an infographic with PiktoChart.

Slides: PPT | Activity Materials

Web Mapping with Carto

Carto is a free web mapping tool. Participants will learn how to make interactive maps, including maps with multiple layers, using various types of data.

Slides: PPT | Instructions | Sample Data

R for Absolute Beginners

This is an introduction to R, a statistical computing software, for participants who have had little or no prior experience with coding.

Slides: PPT / PDF | Instructions | Datasets | YouTube screencast
